# 🎮 Guessing Game

A fun and interactive **Word Guessing Game** built using **HTML, CSS, and JavaScript**. Players must identify the correct word from a scrambled version before the timer runs out. The game includes multiple difficulty levels, score tracking, and a local leaderboard.

## ✨ Features

- 🎯 Three Difficulty Levels
  - Easy
  - Medium
  - Hard
- 🔀 Random scrambled words every round
- ⏳ 20-second timer for each word
- 🏆 Score tracking system
- 📊 Local leaderboard (Top 10 scores)
- 💾 Leaderboard stored using Local Storage
- 🌈 Animated gradient background
- ✨ Neon glowing title animations
- 📱 Responsive and lightweight design
- 🎮 30 levels per game

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Local Storage API

## 🚀 How to Play

1. Open the game in your web browser.
2. Enter your name.
3. Select a difficulty level.
4. Click **Start Game**.
5. A scrambled word will appear.
6. Type the correct word and press **Enter**.
7. Earn **10 points** for every correct answer.
8. Finish all levels or complete the game to view your score.
9. Check your ranking in the leaderboard.

## 🎮 Game Rules

- Each correct answer awards **10 points**.
- Each round has a **20-second** time limit.
- If time runs out, the game automatically moves to the next level.
- There are **30 rounds** in total.
- The leaderboard saves the **Top 10** highest scores on the current device.
